## Overview

Sandboxie Plus gives Windows users a dedicated isolated space for software that is untrusted, newly acquired, or simply better kept separate from the primary system. By containing programs inside a sandbox, it helps limit how far changes can spread and lets you try out apps without merging those effects into your everyday environment.

It also fits common safer-browsing and download scenarios where you want the session itself to remain more compartmentalized. With support for multiple sandboxes and quick removal of sandbox contents, it offers a practical way to handle temporary activity, testing, and containment-oriented workflows on Windows 10 and 11.

## Usage

Typical workflow:

1. Start Sandboxie Plus.
2. Create or select a sandbox.
3. Launch the app, browser, or file you want to isolate inside that sandbox.
4. Use the session normally while it stays separated from the main system.
5. Delete the sandbox contents when you want to discard the changes.

Example workflow:
- Use one sandbox for general untrusted apps
- Use another for browser sessions or downloads
- Clear a sandbox after testing is finished

## FAQ

**How do I get updates?**  
Download the latest build from the linked release location and replace or update your current installation as needed.

**Can I use more than one sandbox?**  
Yes. The tool supports multiple sandboxes so you can separate different tasks.

**Where are my changes stored?**  
Changes stay inside the sandbox until you remove the sandbox contents or otherwise clear the session.

**What if an app does not behave as expected?**  
Check the sandbox you selected, review the configuration, and retry the launch in a fresh sandbox if needed.

**Is there a recommended way to clean up after testing?**  
Delete the sandbox contents after the session to remove temporary changes from that sandbox.
